STUDIO GHIBLI API

This project, carried out by members of OpenBootcamp as part of the 'OpenWeekapps' initiative, focused on developing an API to access data related to movies from the renowned animation studio, Studio Ghibli.

My involvement revolved around backend development for the API. We utilized Express, a web application framework for Node.js, to create routes and handle HTTP requests. The database was implemented using MongoDB, enabling storage and retrieval of information about movies, characters, and associated details within Studio Ghibli productions.
